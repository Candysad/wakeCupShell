# wakeCupShell

## a fake shell program

### 项目组成员

- candySad
- @Cloudkkk
- WakeUp



### 项目名来源

来自于项目组成员每个人提议中的一个词

- candySad：Cup shell

- C1oud：Fuck shell

- WakeUp：wake up shell

  

### 说明

- 一个Linux环境下用纯C编写的伪shell；
- 命令行借鉴bash使用了readline；
- 外部命令可生成可执行文件后放入exCmd；
- 内部命令写在incmd.c中；
- 项目通过makefile写了一个sh脚本tomake.sh用于编译；
